mwl2matlab Toolbox

tt2mat

usage:
ttdata = tt2mat( filename, [option string], [index] )

description:
import spike waveforms from the specified file
[option string] and [index] can be used to select certain waveforms, by filtering on record index, spike index or time. See record filtering for more information.

output:
ttdata.timestamp is the time vector in seconds.
ttdata.spikeid is the id of the record in the tt-file
ttdata.waveform is a WxCxN matrix, W is the number of samples in waveform, C is the number of electrodes, and N is the number of spikes.
ttdata.info is
 a structure containing tt-file parameters
