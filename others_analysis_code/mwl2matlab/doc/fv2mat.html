mwl2matlab Toolbox

fv2mat

usage:
features = fv2mat( filename, [features], [option string], [index] )

description:
imports spike features from the specified file
[features] is a vector of features names that one wants to load. Special strings are: 'all' (include all features, which is the default if [features] is omitted), 'amp' (include the features: 'id', 't_px', 't_py', 't_pa', 't_pb') and 'pos' (include the features: 'pos_x' and 'pos_y')
[option string] and [index] can be used to select certain spikes, by filtering on record index, spike index or time. See record filtering for more information.

output:
features.featurenames contains the features
features.featuredata is a NxM matrix containing N samples and M features
features.info contains additional information about the file
