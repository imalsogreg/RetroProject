mwl2matlab Toolbox

eeg2mat

usage:
eegdata = eeg2mat( filename, [channels], [option string], [index] )

description:

import eeg data from specified file
[channels] argument is an optional vector specifying the channel numbers from which to load data
[option string] and [index] can be used to select part of the eeg data, by filtering on time. See record filtering for more information.

output:

eegdata.time is the time vector in seconds.
eegdata.data is a NxM matrix containing N samples and M channels
eegdata.info is a structure containing eeg parameters
