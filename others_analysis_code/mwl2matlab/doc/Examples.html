mwl2matlab Toolbox

Examples:

eegdata = eeg2mat( 'abcd12.eeg' , [1 2], 'bs|', [1 1.1] )

Retreives 2 channels of data from abcd12.eeg. The data is 0.1 seconds long and the data starts 1 second after the first recorded time in the file.


p = pos2mat( 'pos12.pos', 'r', [100 200] )

This example retrieves 101 frames starting at frame 100.


ttdata = tt2mat( 'abcd12.tt', 'ia', [1:2:1000] )

Retrieves every other spike in the waveform file within the first 1000 records.


pdata = p2mat( 'pos12.p', 's|>', [200 100] )

This example retrieves 100 seconds of diode position data starting 200 seconds after the first timestamp in the file.


features = fv2mat( 'feat.pxyabw', {'pos_x','pos_y'}, 's', [101.5 115.5] )

This retrieves the id and the position of spikes between 101.5 to 115.5 seconds.
