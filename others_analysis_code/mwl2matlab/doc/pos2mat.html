mwl2matlab Toolbox

pos2mat

usage:
posdata = pos2mat( filename, [option string], [index] )

description:
import raw position (i.e. thresholded pixels) from specified file
[option string] and [index] can be used to select certain position records, by filtering on record index or time. See record filtering for more information.

output:
posdata.nitems is the number of threshold crossings in the frame
posdata.frame is the id of the frame
posdata.timestamp is the timestamp array in seconds
posdata.pos is a structure array with the x,y coordinates
posdata.info is the structure containing info about position file.
